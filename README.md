# House Price Prediction using Machine Learning

This project predicts house prices using machine learning models.

## Dataset
Kaggle Competition: House Prices - Advanced Regression Techniques

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Machine Learning Models
1. Linear Regression
2. Random Forest Regressor

## Project Workflow
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Feature Selection
6. Train-Test Split
7. Model Training
8. Model Evaluation using RMSE
9. Model Comparison

## Results
Random Forest performed better than Linear Regression with lower RMSE.
